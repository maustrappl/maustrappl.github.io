Test ist ein methodischer Versuch, mit dem festgestellt werden soll, ob Eigenschaften oder Leistung einer Sache, einer Person oder einer Hypothese den Erwartungen entsprechen.[1][2] Verschiedene Wirtschaftszweige verwenden nicht deckungsgleiche Definitionen. So steht Test teilweise synonym für Qualitäts- oder Endprüfung, Erprobung oder Messung. Die Standardisierungsorganisationen ISO und IEC definieren den Test im Sinne einer Ermittlung.[2] Die DIN 1319 verwendet hingegen die Bezeichnung Prüfen, bei dem festgestellt wird wie weit ein Prüfobjekt die Forderungen erfüllt.[3] In der Technik wird ein Test häufig als Verifikation verstanden, d. h. ob sich ein Testobjekt unter möglichst realistischen Bedingungen wie erwartet verhält. Je wirklichkeitsnäher die Testbedingungen sind, desto aussagekräftiger ist der Test. Nicht immer ist es möglich einen Test unter Realbedingungen durchzuführen, so müssen die Bedingungen möglichst realistisch simuliert werden.[4] Während im Extremfall ein Test einen Beweis für das Vorliegen von Fehlern liefern kann, kann Fehlerfreiheit durch einen Test nicht bewiesen werden.[5] In der disziplinübergreifenden Qualitätsprüfung ist darunter vor allem der statistische Test gemeint. Dabei wird mit Stichproben die Gültigkeit von Hypothesen überprüft.[2] In diesem Sinne wird Test z. B. bei technischen Produkten bzw. Prozessen,[6] bei der medizinischen Diagnostik[7] oder der Wissenschaft[8] verwendet. Der Test unterscheidet sich vom Experiment dadurch, dass es beim Test eine Erwartung gibt, die belegt beziehungsweise widerlegt werden soll, während das Ergebnis beim Experiment offen ist oder nur vermutet werden kann.[9] (Quelle: wikipedia
